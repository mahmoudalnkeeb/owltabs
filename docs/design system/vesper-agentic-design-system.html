Trust-tier legend

Every component that represents an agent action carries one of three colors. This is the only semantic color system in the kit — learn it once, read it everywhere.

Mint — read-only, reversible by definition, never prompts
Amber — mutates state, always confirmable
Red — destructive / irreversible, always confirmed

01 · Foundations

Motion

Motion is how this system tells the truth about time. An agent that takes 4 seconds to think should never look the same as one that's stuck — durations and easings are tuned so "still working" and "done" are unmistakable, even out of the corner of your eye.

Token Value Use
dur-instant 80ms Press/active feedback
dur-fast 120ms Hover, focus ring
dur-base 200ms Default transition (color, opacity)
dur-slow 320ms Panel/drawer open, orb state change
dur-deliberate 480ms Permission card reveal — slow enough to register as "this needs you"
ease-standard cubic-bezier(.4,0,.2,1) Default for everything
ease-breath cubic-bezier(.45,0,.55,1) Orb glow pulse — symmetrical, calm
Two motifs that recur everywhere
PULSE — used for listening / thinking / acting
Glow breathes in and out on a 1.2–2.2s loop. Speed = urgency.
REVEAL — used for streamed text
Tokens fade up in small clusters, not character-by-character.

Reduced motion: all loops collapse to a single static state-color swap (no spin, no pulse). Respected globally via prefers-reduced-motion.

02 · System

Agent state model

Six principles, one state machine. Everything in the agentic components section below is an expression of this model — learn it once and every screen in the extension reads the same way.

1 · Show the work
Plans and tool calls are visible by default, collapsible — never hidden behind a single "thinking…" spinner.
2 · Every mutation is confirmed
Trust-tier color decides whether an action needs your sign-off before it runs.
3 · One glanceable status
The orb is the single source of truth for "what is it doing right now," legible from across the room.
4 · Honest latency
No spinner ever lies. Stream tokens as they arrive; name what step is slow.
5 · Idle is quiet
With nothing running, this is just a fast homepage — search, shortcuts, widgets. The agent recedes.
6 · Always interruptible
A stop control is reachable in every active state, never buried in a menu.
The eight states

Click a state below — it drives the orb, the status copy, and (where relevant) the example tool list.

Agentic components

The components that make this an agent surface rather than a search page. Each one expresses the trust-tier and state-model rules above in a specific UI shape.

Prompt / command bar

Same control as the idle omnibox, but its anatomy is fixed: agent avatar (state), input, mode pills, context chips, send. Mode switches what "enter" does — search vs. delegate to the agent.

Thinking disclosure

03 · Reference

Accessibility

An agent that narrates itself only visually has failed half its users. Every status here is also exposed in text and structure.

Contrast — text on surface-0 (#101010)
Token Hex Ratio Result
text-primary #FFFFFF 19.0:1 AAA
text-secondary #A0A0A0 7.3:1 AAA
text-tertiary #7E7E7E 4.7:1 AA (body) — labels/captions only
accent #FFC799 12.6:1 AAA
mint #99FFE4 16.1:1 AAA
red #FF8080 7.8:1 AAA
#000 on accent button label 13.9:1 AAA
Never color-only

Trust tier is color + a left-edge position + the word "read-only" / "needs approval" in the badge text. Status icons use shape (circle outline → spinner ring → check / × ) in addition to mint/red.

Live regions

The orb's state label and the streaming text container are both aria-live="polite", so a screen reader announces "Thinking… Acting… Done" without re-reading the whole transcript.

Focus

2px accent outline, 2px offset, on every interactive element — including inside the agent drawer and permission prompt, which trap focus while open.

Reduced motion

All spins, pulses and reveals collapse to instant state swaps under prefers-reduced-motion: reduce. Nothing here is motion-only.
